**AccuraSol Automation Assessment**

This repository presents a comprehensive end-to-end (E2E) test automation framework developed using Playwright, designed to validate both UI flows and API functionality.

**Key Features**

• UI Automation Coverage
Automates core user journeys including authentication, product selection, cart verification, and checkout completion.

• API Validation
Verifies real-time weather data using the OpenWeatherMap API, ensuring correct response status and data accuracy.

• Page Object Model (POM) Architecture
Implements a modular POM structure to promote clean code, reusability, and easy maintenance.
